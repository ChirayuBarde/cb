Faber Kitchen Chimney Project was development of various Control Panel for new feature to throw fresh &
cool air to the user while cooking food, so that user do not suffogate while cooking.
The aim of this project is to rotate 24V dc Fan with three different speed.
Three different types of control panel was desgin, they are as follows
Knob based Control Panel which include Voltage Regulator IC with Variable Resistor to achieve different speed.
Button based Control Panel which include Microcontroller IC with two Push to ON button to achieve three different speed.
Touch based Control Panel which include Microcontroller IC with inbuilt touch button to achieve three different speed.
